# EduQuest — Описание правок

## Запуск

```bash
cd EduQuest
uv sync          # установит зависимости
uv run python server.py
# Открыть http://127.0.0.1:5000/
```

База данных создаётся автоматически в `DB/eduquest.db` со всеми категориями,
достижениями и значками при первом запуске.

> **Важно**: ваш файл `Assets/Logos/Logo.png` не входил в архив, скопируйте его
> в `EduQuest/Assets/Logos/Logo.png` после распаковки.

---

## 1. Лидеры
- Колонки: **Место**, **Пользователь** (с аватаркой), **Уровень**, **Рейтинг**,
  **Выполненные материалы**, **Созданные материалы**.
- Числовые ячейки выровнены по центру (`col-center`).
- В колонке «Уровень» отображается номер уровня и общее количество XP:
  `Уровень N · X XP`.
- Сверху вкладки сортировки: Уровень / Рейтинг / Выполненные / Созданные.
- **Рейтинг** = `XP + completed × 25 + created × 100`
  (формула в `database.py::compute_rating`).
- Уровень считается как `XP // 100 + 1` (`XP_PER_LEVEL = 100`).

## 2. Шапка
- Из шапки убраны «Обучение» и «Лидеры».
- Добавлена «Моё обучение» → `/learning`, отдельная страница с разделами
  «Не закончены» и «Завершены».
- Гамбургер-меню переехало в правую часть.
- В правом меню по порядку: Настройки профиля, Моё обучение, Лидеры, Настройки,
  Политика, Поддержка, Выход.

## 3. Главная страница
- Только 3 главных категории: **Знания**, **Навыки**, **Развитие**.
- У каждой свои подкатегории (см. `DEFAULT_CATEGORY_TREE` в `database.py`).
- Чипы «В тренде» / «Новые», далее `select` категории и (если выбрана
  родительская) `select` подкатегории.
- Создание материала использует те же селекты.

## 4. Материалы
- Автор может **редактировать** (`/material/<id>/edit`) и **удалять**
  (модальное окно с «Отмена» / «Удалить»).
- Счётчик просмотров не накручивается: уникальный просмотр на пару
  `(user_id, material_id)` в таблице `material_views_log`.
- 5-балльная оценка под комментариями (`material_ratings`).
- В комментариях `white-space: pre-wrap` + кнопка «Расширить» для длинных
  комментариев (обрезка до 4 строк).
- Кнопка «К списку материалов» удалена.
- Прохождение по шагам: **Начать** → пошаговое отображение → **Далее** →
  на последнем шаге **Завершить материал**, после чего начисляется `+50 XP`
  и пересчитываются достижения/значки.

## 5. Профиль
- 12 достижений (обобщённых, не по конкретным темам).
- 9 значков (Новичок / Ученик / Мастер / Любознательный / Знаток / Создатель /
  Производитель / Стартовый темп / Постоянство).
- В профиле отображаются Уровень, XP, Рейтинг, Место, Созданные материалы,
  Выполненные материалы и прогресс уровня.

## 6. Общее
- `/profile/settings` и `/settings` — заглушки.
- `/policy` — заполненная политика конфиденциальности (6 разделов).
- `/support` — заглушка.
- Достижения «Автор» (1+ материал) и «Наставник» (5+ материалов) теперь
  выдаются автоматически при создании материала через
  `db.recalc_user_awards(user_id)`. Также проверяется при завершении
  материала и публикации комментария.

## Достижения (12)
1. Первые шаги — 1 завершённый материал
2. Уверенный учащийся — 10 завершённых
3. Эксперт — 50 завершённых
4. Автор — 1 созданный материал
5. Наставник — 5 созданных материалов
6. Хранитель знаний — 20 созданных
7. Целеустремлённый — серия 7 дней
8. Марафонец — серия 30 дней
9. Активный участник — 25 комментариев
10. Восходящая звезда — уровень 5
11. Легенда — уровень 20
12. Чемпион — топ-10 рейтинга (минимум 10 пользователей)

## Значки (9)
1. Новичок — уровень 3
2. Ученик — уровень 10
3. Мастер — уровень 25
4. Любознательный — 5 завершённых
5. Знаток — 25 завершённых
6. Создатель — 3 созданных
7. Производитель — 15 созданных
8. Стартовый темп — серия 3 дня
9. Постоянство — серия 14 дней
